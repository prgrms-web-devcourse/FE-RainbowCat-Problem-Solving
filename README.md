## 진행 방법

### 매일 해야 하는 것

- 매일 올라오는 슬랙 리마인더에 스레드로 푼 문제 인증하기
- 인증은 자신이 푼 문제에 대한 풀이방법을 간단히 요약

- **금요일 10시** 디스코드 스터디룸 2에서 모의고사 진행
